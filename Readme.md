# 📘 Miniguia de Estudo – Educação Financeira para Iniciantes

## 🎯 Objetivo
Compreender conceitos fundamentais de educação financeira utilizando IA como ferramenta de apoio.

## 📚 Fontes Utilizadas
- Cartilha do Banco Central
- Material do Sebrae
- Artigo sobre Juros Compostos
- PDF Universitário sobre Orçamento

## 🧠 Metodologia
Utilização do NotebookLM para:
- Organizar fontes
- Criar resumos estruturados
- Elaborar glossário
- Testar variações de prompts
- Analisar respostas com base nas referências

## 📌 Principais Aprendizados
- Importância da reserva de emergência
- Impacto dos juros compostos
- Planejamento financeiro estratégico
