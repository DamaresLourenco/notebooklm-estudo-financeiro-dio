# 📘 Miniguia de Estudo – Educação Financeira para Iniciantes

> 💡 Este repositório faz parte do desafio da DIO sobre uso do NotebookLM como ferramenta de aprendizagem ativa.

---

## 🎯 Objetivo
Compreender conceitos fundamentais de educação financeira utilizando IA como ferramenta de apoio.

---

## 📚 Fontes Utilizadas
- Cartilha do Banco Central
- Material do Sebrae
- Artigo sobre Juros Compostos
- PDF Universitário sobre Orçamento

---

## 🧠 Metodologia
Utilização do NotebookLM para:
- Organizar fontes
- Criar resumos estruturados
- Elaborar glossário
- Testar variações de prompts
- Analisar respostas com base nas referências

---

## 📌 Principais Aprendizados
- Importância da reserva de emergência
- Impacto dos juros compostos
- Planejamento financeiro estratégico

## 🤖 Como Utilizar os Prompts

1. Acesse o Notebook pelo link acima.
2. Vá até a área de chat.
3. Copie um dos prompts abaixo.
4. Cole no campo de pergunta.
5. Analise a resposta e observe as referências utilizadas pela IA.

---

## 🔁 Prompts Reutilizáveis

### 📘 Para Resumos
- Resuma o conteúdo em tópicos principais.
- Explique o tema de forma simples e objetiva.
- Faça um resumo estruturado em introdução, desenvolvimento e conclusão.

### 📊 Para Comparações
- Compare juros simples e juros compostos.
- Diferencie despesa fixa e despesa variável.
- Explique vantagens e desvantagens da reserva de emergência.

### 🧠 Para Revisão
- Crie 5 perguntas para revisar este conteúdo.
- Explique como se estivesse ensinando para um iniciante.
- Gere um exemplo prático com números reais.

---

## 🎯 Objetivo dos Prompts

Esses prompts foram testados durante o projeto para:

- Melhorar a compreensão do conteúdo
- Explorar diferentes formas de explicação
- Estimular pensamento crítico
- Validar informações com base nas fontes

Sinta-se à vontade para adaptar os prompts conforme sua necessidade.
